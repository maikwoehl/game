The graphics should be loaded in cache, if you open this page the first time or after a cache trashing. For this you select Map 0 and click on "Select Map" a few times (ca. 3x).
If you press [G] you can overlay a grid to see the tiles. Press [P] do add a player. The standard field is Map1. If you want to change the map, you can select it above.
